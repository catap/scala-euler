# scala-euler
Another solver for Project Euler on scala.

Just clone it and rub by `sbt run` for solve all solved problem.

Meanwhile you can specified a problem number by `sbt "run 1 7 6"`.

For example:
```bash
➜  euler git:(master) sbt "run 1 7 6"
```
